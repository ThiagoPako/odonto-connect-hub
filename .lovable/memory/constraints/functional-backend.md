---
name: Backend funcional obrigatório
description: Todo botão/função no frontend DEVE ter implementação funcional no backend — nada de UI vazia
type: constraint
---
Nunca criar botões, ações ou funcionalidades no frontend que não tenham implementação real no backend. **Why:** O usuário exige que tudo seja funcional, não apenas visual. Antes de adicionar qualquer interação na UI, garantir que o endpoint/API correspondente existe e funciona.
